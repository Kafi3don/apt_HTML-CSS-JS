ORDERED LISTINGS ; using the ≺ol≻ tag and the type and start attributes.

We shall do some ordered listing of fruits; mango, paw-paw,orange,banana and apple.

Ordered listing means that a list will be numbered from 1 to n (unless otherwise specified using the type and start attributes.
The type attribute enables us to use roman numerals or even alphabet numbering. WHILE the start attribute and its value enables us to state where the numbering starts from.
